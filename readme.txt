IG Portfolio is a clean and easy-to-use portfolio plugin for WordPress. 

== Description ==
Add your projects and display them on a specific page using our template system, shortcodes and widgets.

= IG Portfolio Premium =
* [IG Portfolio Premium](http://themes.iografica.it/downloads/ig-portfolio-premium) add advanced features and give access to priority support.
* Custom portfolio page template
* Metadata customization

== Installation ==
1. Download the plugin.
2. Unzip the folder into the /wp-content/plugins/ directory.
3. Activate the plugin through the Plugins menu in WordPress.
